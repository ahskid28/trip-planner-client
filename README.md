
# Smart Travel Planner

AI-powered travel planning application built with Next.js and Tailwind CSS.

## Features

- User Authentication
- Trip Creation
- Itinerary Management
- Budget Tracking
- Expense Tracking
- AI Travel Assistance

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL

## Live Demo

https://trip-planner-client-eight.vercel.app/login

## Run Locally

npm install
npm run dev
